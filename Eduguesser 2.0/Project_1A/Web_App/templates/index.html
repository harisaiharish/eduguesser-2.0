What is Eduguesser?



Eduguesser is an educational game with the goal of raising the importance of general geographic, cultural and historical awareness, using fun facts and mesmerizing pictures. To play the game, the user has to select a country on the map based on the information given to them, and is awarded points based on how close they are to the actual country.
